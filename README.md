### PokeAPI

#### Get Pokemon

```http
  GET https://pokeapi.co/api/v2/pokemon/{id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of pokemon to fetch |
